# Public Portfolio Plan

## Positioning

Gun is an IT Infrastructure & Cloud Architect with a separate personal AI Engineering Lab. The portfolio should clearly separate professional infrastructure work from personal AI engineering R&D: orchestration, harness engineering, long-running development windows, and app development evidence.

## Content pillars

1. **Professional Infrastructure Work** — career evidence from infrastructure, cloud, M365, security, migration, and operations.
2. **AI Engineering Lab** — personal R&D track for AI-assisted apps, orchestration workflows, and automation.
3. **Harness Engineering** — differentiates the workflow and review/governance approach.
4. **Long Work Window Playbook** — shows disciplined supervised execution for complex, multi-hour work.
5. **Security and Sanitization** — shows responsible public release practice.

## Recommended publish sequence

1. Reframe gunkr.com around two tracks: Professional Infrastructure Work and AI Engineering Lab.
2. Keep AzVision as a concrete AI Engineering Lab / app-development case study.
3. Publish sanitized Long Work Window and Harness Engineering writeups for technical feedback.
4. Use Reddit/community feedback to refine claims before LinkedIn.
5. Add a short LinkedIn featured post after wording has been validated.
