# Security Policy

This repository is a sanitized public case-study and lab-notes surface. It should not contain credentials, private runtime data, production configuration, real cloud exports, raw logs, or account secrets.

If you notice accidental sensitive data exposure in this repository, please open a GitHub issue with a minimal description of the affected file and line. Do not include the sensitive value itself in the issue.

Security issues in private infrastructure, live services, or hosting accounts are outside the scope of this public repository.
